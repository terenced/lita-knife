# lita-knife
A chef/knife wrapper [Lita](https://www.lita.io/).
Port of [hubot-chef](https://github.com/hubot-scripts/hubot-chef).

## Installation

Add lita-knife to your Lita instance's Gemfile:

``` ruby
gem "lita-knife"
```
